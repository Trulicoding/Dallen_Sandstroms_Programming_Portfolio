CS2350: Client-side Web Development

HTML5

HTML5 is the 5th major version of HTML. HTML stands for Hypertext Markup Language and is used on the world wide web as a tool to present content.

CSS3

CSS3 is the third variation of the Cascading Style Sheets language. It is used alongside markup languages to describe the presentation of the content written in a markup language.

JavaScript

Javascript is used commonly on websites for writing webpage behavior.
